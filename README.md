# Calculadora-Investimentos
Calculadora de investimentos no Excel para projeto da DIO

Fiz uma calculadora de investimentos usando o excel assim como fui instruido nas video aulas. Funciona como o esperado, calculando sugestão de investimento, patrimônmio acumulado, dividendos e quanto será investido em cada tipo de fundo imobiliário. Além de um gráfico ao final deixando mais visual esse percentual investido. Resolvi deixar tudo centralizado Para um visual mais mobile, pensando que tal tabela possa ser transformada em uma aplicação.
